# WakeBet

### 제때에 일어나지 않으면 돈을 잃는 기상 습관 형성 서비스

## 프로젝트 소개

WakeBet은 사용자가 설정한 기상 시간에 맞추어 일어나도록 돕는 웹 서비스다.

이 프로젝트는 **손실 회피(Loss Aversion)** 개념에서 아이디어를 얻어, 
사용자가 스스로에게 금전적 책임을 부여하고, 이를 통해 기상 습관을 형성할 수 있도록 설계되었다.

본 프로젝트는 대학 교양 수업 *SW 프로그래밍의 기초* 과제용으로 제작되었으며, 실제 결제 시스템이 아닌 가상 화폐를 사용한다.

---

## 문제 정의

많은 사람들은 중요한 일정이나 학업, 업무를 위해 정해진 시간에 일어나야 하지만 알람을 끄고 다시 잠드는 경우가 많다.

WakeBet은 다음과 같은 문제 의식을 갖늗다.

> "만약 늦잠을 자는 행위가 돈을 잃는 것으로 이어진다면, 알람에 맞춰서 일어나는 것에 대한 강제성이 생기지 않을까?"

---

## 대상 사용자

* 특정 시간에 반드시 일어나야 하는 사람
* 아침 기상 습관을 만들고 싶은 사람
* 알람만으로는 기상에 어려움을 겪는 사람

---

## 핵심 기능

### 1. 기상 목표 설정

사용자는 다음 정보를 설정할 수 있다.

* 기상 시간
* 실패 시 지불할 예약금(Deposit)

예약금을 설정하면 사용자의 가상 지갑에서 해당 금액이 차감된다.

---

### 2. 기상 인증 게임

사용자가 기상 시간 이후 서비스에 접속하면 여러 개의 기상 인증 게임을 수행해야 한다.

예시:

* 기억력 게임 
* 계산 게임
* 반응 속도 게임 
* 단어 거꾸로 쓰기 게임
* 일치하지 않는 것 하나 찾기 게임 

각 게임은 Pass 또는 Fail로 평가된다.

---

### 3. 기상 성공/실패 판정

게임 수행 결과를 기반으로 성공률을 계산한다.

* 일정 기준 이상 통과 → 기상 성공
* 기준 미달 → 기상 실패

---

### 4. 예약금 처리

#### 기상 성공

* 예약금 전액 반환

#### 기상 실패

* 예약금 몰수

실제 서비스에서는 특정 개인 또는 단체에 송금하는 기능을 고려할 수 있으나, 본 프로젝트에서는 구현하지 않았다.

---

## 동작 과정

1. 사용자가 기상 시간을 설정한다.
2. 사용자가 예약금을 설정한다.
3. 예약금이 가상 지갑에서 차감된다.
4. 기상 시간 이후 인증 게임을 수행한다.
5. 게임 결과를 평가한다.
6. 성공 시 예약금을 반환한다.
7. 실패 시 예약금을 몰수한다.

## 프로젝트 목표

본 프로젝트의 목표는 다음 개념을 웹 서비스 형태로 구현하는 것이다.

* Loss Aversion (손실 회피)
* Commitment Device (자기 구속 장치)
* Habit Formation (습관 형성)

---

## 한계점

본 프로젝트는 학습 목적의 프로토타입이다.

따라서 다음 기능은 포함하지 않는다.

* 실제 결제 시스템
* 실제 환불 시스템
* 실제 계좌 송금
* 실제 알람 기능
* 서버 기반 사용자 인증

모든 금액은 가상 화폐(Credits)로 처리되며, 데이터는 브라우저 LocalStorage에 저장된다.
